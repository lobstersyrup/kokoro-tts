# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please open a GitHub Issue with the label `security` or contact the maintainer directly.

This server is designed to run locally (`localhost:8880`) and should not be exposed to the public internet without authentication middleware in front of it.
